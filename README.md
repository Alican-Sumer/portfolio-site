## Pushing to GitHub

This project is already a Git repo with an initial commit on the `main` branch. To put it on GitHub:

1. **Create a new repository on GitHub**
   - Go to [github.com/new](https://github.com/new)
   - Choose a name (e.g. `portfolio`), leave "Initialize with README" **unchecked**
   - Create the repository

2. **Add GitHub as remote and push**
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git push -u origin main
   ```
   Replace `YOUR_USERNAME` and `YOUR_REPO_NAME` with your GitHub username and repo name. If you use SSH:
   ```bash
   git remote add origin git@github.com:YOUR_USERNAME/YOUR_REPO_NAME.git
   git push -u origin main
   ```
